# TOC 消费机器人 · AIPL 人群-数据-产品飞轮

一个面向消费机器人品类的 AIPL 营销方法论拆解页面，覆盖前付费链路（A+I）与后转化链路（P+L），并融入 5 大核心漏斗、6 维埋点体系与数据反哺硬件迭代的完整闭环。

## 在线预览

本仓库可直接部署到 GitHub Pages，部署后访问：

```
https://<你的用户名>.github.io/<仓库名>/
```

## 页面内容

- **AIPL 横向链路**：A → I → P → L 一条线贯穿
- **前付费链路（A+I）**：种草漏斗 + 体验漏斗，从 0→1 到 1→100 的阶段重点
- **后转化链路（P+L）**：激活漏斗 + 留存推荐漏斗，从 0→1 到 1→100 的阶段重点
- **数据反哺飞轮**：埋点 → 标签 → 洞察 → 人群校准 → 硬件迭代 → 验证
- **六维埋点体系**：用户 / 行为 / 场景 / 设备 / 内容 / 性能
- **全链路指标看板**：5 漏斗转化率总览 + 健康度雷达 + 断崖流失分布

## 技术说明

- 纯静态 HTML，无需构建工具
- 使用 ECharts 绘制数据图表
- 使用 Mermaid 绘制架构图
- 响应式布局，支持桌面端和移动端

## 本地预览

直接在浏览器中打开 `index.html` 即可预览。

## 部署到 GitHub Pages

1. 在 GitHub 创建一个新仓库
2. 将本文件夹内所有内容推送到仓库主分支
3. 进入仓库 **Settings → Pages**
4. Source 选择 **Deploy from a branch**，分支选择 **main**，文件夹选择 **/(root)**
5. 保存后等待 1-2 分钟即可通过 `https://<你的用户名>.github.io/<仓库名>/` 访问
