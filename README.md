# AIPL 消费机器人飞轮页面

可直接部署到 GitHub Pages 的单文件版本。

## 部署步骤

1. 把本文件夹内所有文件上传到你的 GitHub 仓库根目录
2. 打开 GitHub 仓库 **Settings → Pages**
3. Source 选择 **Deploy from a branch**
4. 分支选择 **main**，文件夹选择 **/(root)**
5. 保存后等待 1-2 分钟
6. 访问 `https://<你的用户名>.github.io/<仓库名>/`

## 文件说明

- `index.html`：完整页面（单文件，包含内联 CSS 和图表脚本，依赖 CDN）
- `README.md`：本说明文件

无需其他依赖，直接部署即可。
